# Mini Express

A lightweight Express-like backend framework built from scratch using Node.js' native `http` module.

This project was created as a learning exercise to understand how web frameworks such as Express work internally.

## Features

- HTTP Server using Node.js `http` module
- Route Registration
- GET Routes
- POST Routes
- Middleware System
- Custom Logger Middleware
- JSON Body Parser Middleware
- 404 Route Handling
- Centralized Error Handling
